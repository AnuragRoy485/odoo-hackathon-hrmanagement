HRMS Authentication

Employee login, staff onboarding, and password control in one place.

This dashboard is wired directly to the DRF auth endpoints. HR and admin users can create employees, send verification links, inspect the current session, and reset passwords. Normal employees can sign in, verify their email, and change the system-generated password on first login.
